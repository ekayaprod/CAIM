Help & Documentation

Overview

This tool creates a single, powerful "Form Filler" bookmarklet to automate complex forms using hierarchical "Preset Groups".

Phase 1: Field Mapping

Use the "Field Mapper Tool" on your target form. As you map fields, they will be highlighted in green on the page for easy tracking.

The updated mapper tool now includes a **"Scan Duallists"** button to automatically map all duallist fields, and a **"Map Key Field"** button to identify the constant field (like a username) used for session validation.

Phase 2: Configuration & Generation

  1. Load Mappings: Paste the JSON from the mapper tool.
  2. Configure Behaviors: Mark any `duallist` field as "Add All". When the final bookmarklet runs, if your Excel preset has *any* value for this field (e.g., "Yes"), it will add all available options instead of searching for a specific one.
  3. Import Preset Groups: Import Excel files. You'll choose how to handle blank cells.
  4. Set Hierarchy: Drag and drop the imported groups to set their merge priority.
  5. Generate Bookmarklet: Create the final "Form Filler" bookmarklet.

How to Use the Form Filler Bookmarklet

The generated bookmarklet is "stateless" and intelligent, making it robust against page reloads and context changes.

  1. First Click: Click the bookmarklet. It will prompt you to select your presets.
  2. The filler will complete all standard fields first, then perform the first action that causes a page reload (like a duallist update) and then stop, showing a progress overlay.
  3. Subsequent Clicks: After the page reloads, simply click the bookmarklet again. It will automatically continue with the next reloading action. If it detects that the key field (e.g., username) has changed, it will restart the process, asking you to select presets again.
