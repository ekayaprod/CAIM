Help & Documentation

Overview

This tool creates a single, powerful "Form Filler" bookmarklet to automate complex forms using hierarchical "Preset Groups".

Phase 1: Field Mapping

Use the "Field Mapper Tool" on your target form to generate a JSON configuration. This tells the filler what fields exist and what type they are.

Phase 2: Configuration & Generation

  1. Load Mappings: Paste the JSON into the text area.
  2. Import Preset Groups: Import Excel files. For each file, you'll be asked how to treat blank cells:
    • Ignore Blanks (Default): Blank cells in your Excel will not affect the corresponding form fields.
    • Blanks Mean 'Clear': A blank cell is an explicit instruction to empty the corresponding form field.
  3. Set Hierarchy: Drag and drop the imported groups to set their merge priority.
  4. Generate Bookmarklet: Create the final "Form Filler" bookmarklet.

How to Use the Form Filler Bookmarklet

The generated bookmarklet is intelligent and handles page reloads caused by the form.

  1. Click the bookmarklet. A modal will appear allowing you to select from your preset groups (e.g., a "Location" preset, a "Role" preset).
  2. After you make your selections and click "Fill", the tool scans the form and builds an optimized list of necessary changes. Simple additions are prioritized to run first.
  3. It then begins applying changes. If a change causes a page reload, a progress screen will appear.
  4. **After the page reloads, simply click the bookmarklet again.** It will automatically resume where it left off.
  5. Repeat until you see the "Fill Complete!" message.
